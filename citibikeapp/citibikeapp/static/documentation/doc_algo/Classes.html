Classes¶

The classes necessary to run the simulator

StationMap¶

class StationMap.StationMap¶

Specifies a map of rebalencing stations

Variables:
  • stations (Dict[int, Station]) – A dictionary that maps the ID of a station in this map to its Station object
  • original (Dict[int,Station]) – A deepcopy of stations when this StationMap was initialized
  • forbidden (Dict[int, Tuple[datetime, datetime]) – A dictionary that maps a Station ID to a tuple of the start/end datetime for when this station is forbidden
add_forbidden_station(station, start, end)¶

Makes a station in this list forbidden for a period of time

Parameters:
  • station (Station) – The station that should be forbidden
  • start (datetime) – The starting time of when this station is forbidden
  • end (datetime) – The ending time of when this station is forbidden
compute_disappointed_customers(time, station, bikes)¶

Finds the number of disappointed customers expected at a certain station with a specified number of bikes

Parameters:
  • time (timedelta) – The time to check expected number of disappointed customers
  • station (Station) – The station that customers are being disappointed by
  • bikes (int) – The number of bikes at this station to examine the effects of
Returns:

The number of disappointed customers caused by having this many bikes at this station at this time.
Return type:

float
create_checkpoint()¶

Makes a checkpoint at this StationMap’s current state. If restore is called after this, the StationMap will revert to the state where this checkpoint was made

find_optimal_number_of_bikes(time, station)¶

Finds the optimal number of bikes that should be at a certain station at a specified time

Parameters:
  • time (timedelta) – The time to check for the number of optimal bikes
  • station (Station) – The station to check for the number of optimal bikes
Returns:

The number of bikes at this station that will minimize the number of disappointed customers
Return type:

int
opt_task(truck, station=None)¶

Finds the optimal task to perform at this station

  • truck (Truck) – The truck that will perform the returned task
  • station (Station) – The station to perform the returned task at
Returns:

The task at this station that will minimize the number of disappointed customers

restore()¶

Reverts this StationMap back to its original state

Truck¶

class StationMap.Truck(origin, location, destination, start=datetime.datetime(1, 1, 1, 0, 0), end=datetime.datetime(1, 1, 1, 0, 0), iden=-1, capacity=42, fill=0, broken_bikes=0, breaks=None, rate=30)¶

Specifies a Rebalencing Truck

Variables:
  • id (int) – The ID number of the truck. Defaults to -1.
  • capacity (int) – The number of bikes that can fit on the truck. Defaults to 42.
  • fill (int) – The number of total bikes (including broken bikes) loaded onto the truck. Defaults to 0.
  • broken_bikes (int) – The number of broken bikes loaded onto this truck. Defaults to 0
  • breaks (list[Break]) – List of Breaks scheduled for this truck. Defaults to empty list.
  • rate (int) – The amount of time in seconds it takes the driver on this truck to move one bike. Defaults to 30 sec.
  • origin (Station) – The Station that this truck started its shift at
  • location (Station) – The Station where this truck finished its last Task
  • destination (Station) – The Station where this truck will return to at the end of its shift
  • start (datetime) – The starting time of this truck’s shift
  • end – The ending time of this truck’s shift
add_bikes(station, bikes, broken)¶

Adds bikes to this truck and removes bikes from the station this truck is taking from

Parameters:
  • station (Station) – The station is truck is taking bikes from
  • bikes (int) – The number of bikes to be added to this truck
  • broken (bool) – True if the bikes being moved are broken, False if they are normal
greedy_move(smap, needs_exit=True)¶

Greedily finds the task with the highest (change in disappointed customers)/(time to perform task) ratio for this truck

Parameters:
  • smap (StationMap) – The StationMap this truck is on
  • needs_exit (bool) – Specifies whether time must be allotted for the truck to return to its destination. Returns True if it needs to return, and False if it doesn’t. Defaults to True.
Returns:

The task with the highest score. Returns None if there are no feasible tasks
Return type:

Task, None
greedy_random_move(smap, k=5, needs_exit=True)¶
Select a task from the top k tasks with the highest (change in disappointed customers)/(time to perform task)
ratio for this truck, where each task has a probability in proportion to 2 to the power of its score.
Parameters:
  • smap (StationMap) – The StationMap this truck is on
  • k (int) – Specifies how many tasks should be on the top k list
  • needs_exit (bool) – Specifies whether time must be allotted for the truck to return to its destination. Returns True if it needs to return, and False if it doesn’t. Defaults to True.
Returns:

The task with the highest score. Returns None if there are no feasible tasks
Return type:

Task, None

Station¶

class StationMap.Station(index, iden=-1, bikes=0, docks=0, rate=120)¶

Specifies a bikeshare station with bikes and docks

Variables:
  • id (int) – The ID number of the station. Defaults to -1
  • bikes (int) – The number of bikes (including broken bikes) currently at this station. Defaults to 0
  • docks (int) – The number of docks at this station that are empty and not broken. Defaults to 0
  • rate (int) – The amount of time in seconds it takes for a truck to park at this station. Defaults to 120

Task¶

class StationMap.Task(station, bikes=0, pickup=True, broken=False)¶

Specifies a rebalencing task for a driver to perform

Variables:
  • station (Station) – The station where bikes should be moved.
  • bikes (int) – The number of bikes to be moved. Defaults to 0.
  • pickup (bool) – Is true if bikes are to be picked up into the truck; is false if bikes are to be dropped off. Defaults to True
  • broken (bool) – Is true if moving broken bikes; if false if moving regular bikes. Defaults to False.

Returns:

True if this task can be completed, and False if it cannot.
